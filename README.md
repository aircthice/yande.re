# yande.re
<H1>萌妹yande.re iOS通用客户端</H1>
<H3>进度</H3>正在开发(80%)
<H3>简介</H3>
    为二次元同人爱好者打造的二次元高质量同人图片应用，每天都会更新大量的高质量同人图。
    该应用可以为二次元爱好者提供高清同人图的浏览、收藏、下载，并提供分级过滤功能。
